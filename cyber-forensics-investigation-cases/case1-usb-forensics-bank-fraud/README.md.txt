# Case 1 – USB Forensics: Bank Fraud Investigation

Forensic analysis of USB image files to identify hidden, encrypted, and fraudulent data.

## Tools
FTK Imager, zsteg, John the Ripper, GPG, Kali Linux

## Findings
- Hidden OpenPGP keys inside PNG images
- Password-protected Excel and PDF
- Cracked password: langley
- Bitcoin wallet QR found

## Conclusion
Evidence confirms steganography, encryption, and anti-forensic activity.
